📖 Lumina Mail 사용 설명서

🚀 Lumina Mail 시작하기

Lumina Mail은 AI 기반의 채팅형 이메일 클라이언트입니다. 기존 이메일을 채팅처럼 읽고 관리할 수 있습니다.

1
이메일 계정 추가

Cmd + , 를 눌러 설정을 열고, 이메일 계정 정보(IMAP/SMTP)를 입력하세요.

2
메일 동기화

계정 추가 후 "동기화" 버튼을 누르면 INBOX의 메일이 자동으로 가져와집니다.

3
대화 선택

왼쪽 메일 목록에서 대화를 클릭하면 채팅 형태로 메일이 표시됩니다.

4
답장 보내기

하단 입력창에 답장을 작성 후 전송 버튼(또는 Enter)을 누르세요. 5초 내 취소 가능합니다.

✉️ 이메일 계정 설정

Cmd + , 로 설정을 열어 계정을 관리하세요.

Gmail 설정 예시

항목값
IMAP 서버imap.gmail.com
IMAP 포트993
SMTP 서버smtp.gmail.com
SMTP 포트587
사용자명your@gmail.com
비밀번호앱 비밀번호 (2차 인증 필요)
💡 Gmail 앱 비밀번호 생성:

Google 계정 → 보안 → 2단계 인증 활성화 → 앱 비밀번호 생성 → 16자리 코드를 비밀번호 칸에 입력

Outlook / Hotmail 설정

항목값
IMAP 서버outlook.office365.com
IMAP 포트993
SMTP 서버smtp.office365.com
SMTP 포트587

항목값

💡 Naver 메일: 네이버 메일 설정 → POP3/IMAP 사용 허용 필요

🔍 검색 & 태그 기능

태그 검색

검색창에 #태그이름 입력 후 , 또는 Enter를 누르면 태그가 칩(뱃지) 형태로 변환됩니다.

🏷️ 개발용 🏷️ 긴급 ← 이렇게 표시됩니다
  • 여러 태그 칩 = AND 조건 (모든 태그를 가진 메일만 표시)
  • 태그 칩 + 일반 텍스트 = 동시 필터링 가능
  • 칩의 × 클릭 또는 Backspace로 제거

수동 태그 추가

대화 선택 → 우측 인텔리전스 패널 → "+ 태그 추가" 버튼 클릭 → 태그명 입력 후 "추가"

필터 버튼

  • 전체 — 모든 메일 표시
  • 안 읽음 — 읽지 않은 메일만
  • 고정 — 핀 고정된 메일만
  • ⭐ 중요 — 중요 표시된 메일만
  • 📋 조치필요 — AI가 감지한 액션 아이템
  • 🎬 미디어 — 첨부파일이 있는 메일

🤖 AI 기능 (Gemini)

⚙️ API 키 설정 필요:

Cmd + , → AI 설정 → Google Gemini API 키 입력

키 발급: aistudio.google.com에서 무료로 발급 가능

AI 요약

대화를 선택하면 우측 인텔리전스 패널에 AI가 핵심 내용을 자동 요약합니다.

AI 번역

외국어 이메일의 경우 "🌐 번역 보기" 버튼으로 한국어 번역을 확인할 수 있습니다.

자동 태그 분류

AI가 이메일 내용을 분석하여 자동으로 카테고리 태그를 부여합니다. (보고서, 긴급, 회의 등)

비용 절감

Regex 기반 사전 분류로 API 호출을 80% 절감합니다. 간단한 분류는 로컬에서, 복잡한 분석만 AI가 처리합니다.

☁️ 클라우드 백업

대용량 첨부파일(10MB 이상)은 자동으로 클라우드에 업로드됩니다.

지원 서비스

  • Google Drive — Google 계정으로 OAuth 연결
  • OneDrive — Microsoft 계정으로 OAuth 연결

설정 방법

1
설정 열기: Cmd + ,
2
클라우드 탭 이동 → Google Drive 또는 OneDrive "연결" 클릭
3
OAuth 인증: 브라우저에서 계정 로그인 후 권한 허용

❓ 자주 묻는 질문

Q. 메일이 동기화되지 않아요

A. 설정에서 계정의 IMAP 서버 정보를 확인하세요. Gmail의 경우 앱 비밀번호를 사용해야 합니다. "연결 테스트" 버튼으로 확인할 수 있습니다.

Q. AI 요약이 작동하지 않아요

A. 설정 → AI 설정에서 Gemini API 키가 올바르게 입력되었는지 확인하세요. "AI 연결 테스트" 버튼으로 확인 가능합니다.

Q. Gmail에서 "앱 비밀번호"는 어떻게 만드나요?

A. Google 계정 → 보안 → 2단계 인증 활성화 → 앱 비밀번호 페이지에서 "메일" 앱을 선택하고 생성하세요.

Q. 보낸 메일을 취소하고 싶어요

A. 메일 전송 후 5초 이내에 화면 하단의 "전송 취소" 버튼을 누르면 전송이 취소됩니다.

Q. 데이터는 어디에 저장되나요?

A. 모든 데이터는 사용자의 로컬 컴퓨터(SQLite DB)에 저장됩니다. 외부 서버로 전송되지 않습니다. (AI 요약 시에만 Gemini API로 이메일 내용이 전송됩니다.)

Q. 여러 이메일 계정을 동시에 사용할 수 있나요?

A. 네, 설정에서 여러 계정을 추가할 수 있습니다. 각 계정의 메일은 하나의 통합 리스트에 표시되며, 수신 계정 이름이 옆에 표시됩니다.
